# `@blue-water-autonomy/html-eslint-parser`

A parser for HTML-ESLint.
It makes AST generated by [ES-HTML-parser](https://github.com/yeonjuan/es-html-parser) compatible with ESLint.

## Install

```bash
$ yarn add -D @blue-water-autonomy/html-eslint-parser
$ npm install -D @blue-water-autonomy/html-eslint-parser
```

## Usage

Set the `parser` property in your ESLint configuration file.

```json
{
  "parser": "@blue-water-autonomy/html-eslint-parser"
}
```
